# VN Hub

Plataforma SaaS de Gestão Comercial e Operacional integrada para agências, consultorias e empresas de serviços recorrentes.

## Recursos Principais

- **Multi-Tenant Isolado**: Arquitetura segura com isolamento de dados por inquilino.
- **Central de Leads**: Kanban completo para recebimento de leads com status, responsáveis e temperatura.
- **Propostas Comerciais**: Geração de propostas dinâmicas em formato PDF ou páginas públicas seguras.
- **Onboarding Automatizado**: Criação de estruturas de entrega integradas com Google Drive e ClickUp.
- **Faturamento e Cobrança**: Integração financeira para controle de boletos, PIX e cartões (Asaas).
- **Aprovação de Publicações**: Fluxo integrado para aprovação de peças de marketing com o cliente final.

## Como Executar

### Pré-requisitos
- Node.js (v18 ou superior)
- npm / yarn / pnpm

### Passos
1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
